What is div ?

Div is a block level element. It is used to create a division or section in an HTML document. It is a container that is used to group elements. It is a block level element that can contain other block level elements.

What is span ?

Span is an inline element. It is used to style a part of the text. It is used to group elements for styling purposes. It is an inline element that can contain other inline elements.
